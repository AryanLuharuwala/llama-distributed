The one-liner, explained

    
The installer one-liner does five things:

    
    
      
  1. Detects OS (uname) + arch + accelerator (nvidia-smi for CUDA, darwin-arm64 for Metal).
    2. 
      
  2. Resolves the latest release tag from the GitHub API.
    3. 
      
  3. Downloads llama-distributed-<tag>-<os-arch-accel>.tar.gz.
    4. 
      
  4. Extracts to a per-user dir (~/.local/share/llama-distributed on Linux, equivalent on Mac/Windows).
    5. 
      
  5. Registers a per-user service (systemd --user, launchd, or Windows Task Scheduler) and execs dist-node --pair.
    6. 
    

    
The dashboard mints a fresh distpool:// deep-link every time you click Add rig. The pair token has a 5-minute TTL.


    
Manual install (no service)

    
If you don't want a background service, run the binary directly. -g 999 offloads every possible layer to the GPU; -g 0 is CPU-only.

    

      
shell


./bin/dist-join --token <POOL_TOKEN> --server pool.llamadist.dev -g 999

    


    
Uninstall

    
Run the platform installer with uninstall (Linux/macOS) or -Uninstall (Windows):

    

      
shell


./scripts/install/linux-install.sh uninstall
./scripts/install/macos-install.sh uninstall
powershell -File scripts\install\windows-install.ps1 -Uninstall

AWS

    
On a Deep Learning AMI (Ubuntu 22.04), NVIDIA drivers + CUDA are pre-installed. The one-liner picks CUDA automatically.

    

      
ec2-user-data.sh


#!/bin/bash
curl -fsSL https://<your-server>/install.sh \
  | sh -s -- --pair 'distpool://pair?token=<TOKEN>&server=wss://<your-server>/ws/agent'

    


    
Google Cloud

    
Use a Deep Learning VM image (common-cu125). Same one-liner works inside the startup script.


    
Azure

    
On Azure ML compute instances, the nvidia-smi check succeeds out of the box — no extra config.


    
RunPod / Lambda

    
Community GPU rentals with pre-installed CUDA. Just paste the one-liner into the startup script field.


    
Troubleshooting

    
The one-liner 404s

    
No release has been cut yet. Either build from source (see quickstart), or cut a prerelease via workflow_dispatch on the GitHub Actions page.

    
Pair token expired

    
Pair tokens live 5 minutes. Regenerate from the dashboard — the card has an inline timer.

    
Firewall blocks port 8080 on LAN

    
The server prints a pre-flight check on startup. Copy the firewall-cmd line it logs and paste it on the control plane host.
